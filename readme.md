# Hollow Point

Awesome platformer.

## Development

### Server running

You can use [`nodemon`](http://nodemon.io) or [`coffee`](http://coffeescript.org/#usage) command for running via Bash.

### Client-side

You should use [`gulp`](http://gulpjs.com) for compile CS sources
